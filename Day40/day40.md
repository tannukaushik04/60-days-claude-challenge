# 🚀 Day 40 – Build Own AI Assistant

## 📌 Overview
Today I build Ascend—an AI-powered Career Assistant that helps job seekers improve their resumes, prepare for interviews, build their personal brand, and plan an effective job search. I built this production-ready AI Assistant from scratch by combining:

- Product thinking
- Prompt engineering
- UI/UX design
- Claude API integration
- Documentation

---

# 🛠 Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- Claude API
- Prompt Engineering

---

# 🤖 Assistant Built

## Assistant Name
**Ascend — AI Career Strategist**

### Purpose
Ascend helps students and job seekers improve their careers by providing AI-powered guidance for:

- Resume & Cover Letter Review
- Mock Interviews
- Job Search Strategy
- LinkedIn & Personal Branding

The assistant provides structured reports instead of simple chat responses, making the experience feel like a premium career platform.

---

# ✨ Features

- Premium modern UI
- AI-powered Resume Analyzer
- ATS-style Resume Score
- Cover Letter Review
- Mock Interview Preparation
- Job Search Planning
- LinkedIn Branding Suggestions
- Structured Reports
- Live Claude API Integration
- Responsive Design
- Documentation Panel
- Production-quality System Prompt
