# Day 36 – Cognitive Pattern Explorer

## 🧠 Project Overview
**Cognitive Pattern Explorer** is an interactive educational web application that helps users explore their natural thinking tendencies through realistic scenarios, drag-and-drop activities, and guided self-reflection.

The application emphasizes self-awareness rather than assessment. It demonstrates that people approach decisions differently depending on context, priorities, and habits. The experience is educational only and does **not** diagnose or evaluate mental health.

---

# 🎯 Features

- Calm and Stress modes
- Interactive scenario-based learning
- Native drag-and-drop activities
- Personalized reflection journal
- Percentage-based thinking tendency breakdown
- Responsive design
- Smooth animations and transitions
- Keyboard accessibility
- Works completely offline
- Educational only (non-clinical)

---

# 🛠️ Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- Native Drag & Drop API
- Responsive Design
- Offline Single-File Architecture

# 💡 Key Learnings

- Learned that people naturally process information in different ways depending on context.
- Explored how analytical, intuitive, reflective, action-first, and overthinking tendencies influence decisions.
- Understood that self-reflection tools should encourage awareness rather than make judgments.
- Practiced building interactive educational experiences using only HTML, CSS, and JavaScript.
- Implemented responsive layouts, drag-and-drop interactions, progress tracking, and personalized reflections in a single offline application.
- Reinforced the importance of accessible, calm, and user-friendly UI design for educational applications.
