Workflow summary

This build takes a portfolio idea from a one-line concept to a deployed React + Python (FastAPI) application. Planning and design compress from days to hours by using Claude for spec-writing and UI scaffolding; the middle stages (backend, frontend, integration) benefit most from an agentic coding tool like Claude Code that can read and edit the whole repo; testing, deployment, and iteration close the loop with lighter-weight, targeted AI use. Budget roughly 35–55 hours end to end for a solo builder working part-time, most of it in the build stage.

Recommended AI stack

Claude (planning & docs) Claude Code (build & refactor) Claude Design / v0 (UI) Copilot (inline tests) ChatGPT (quick lookups, optional)
