# Day 48 – Build The Verdict Engine

## 📌 Project Overview

Today I built **The Verdict Engine**, an AI-powered decision support application that helps users make informed choices using **real, verifiable data instead of assumptions**. Rather than relying on fixed recommendations, the application allows users to assign personalized weights to different decision criteria and instantly generates a ranked comparison based on trusted sources.

Built as a **single offline HTML application** using **HTML, CSS, and Vanilla JavaScript**, the project combines **interactive decision scoring, live ranking updates, transparent research methodology, and source-backed comparisons** into one premium dashboard.

The application emphasizes **transparency by design**, clearly distinguishing sourced facts from estimates, displaying every citation used, and explaining how conflicting information was resolved. This creates a reliable and user-centric decision-making experience while demonstrating how AI can support critical decisions with evidence rather than speculation.

---

## ✨ Features

- Interactive weighted decision comparison
- Live ranking updates based on user-defined priorities
- Source-backed comparison using real, citable data
- Transparent research methodology panel
- Clearly marked sourced vs estimated values
- Professional responsive dashboard
- Collapsible "How This Was Researched" section
- Dynamic score calculation engine
- Premium dark-themed UI
- Built using pure HTML, CSS, and JavaScript (No frameworks)

---
