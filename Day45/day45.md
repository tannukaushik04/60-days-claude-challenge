# Day 45 – AI Decision Strategist

## 📌 Project Overview

Today I built **AI Decision Strategist**, an AI-powered decision support application that helps users make complex life and career decisions through structured reasoning instead of emotional guesswork.

The application guides users through a short interview, evaluates multiple options using decision intelligence frameworks, identifies cognitive biases and hidden assumptions, generates a weighted decision matrix, performs a premortem risk analysis, creates a 7-day validation plan, and delivers a professional interactive decision report.

Built as a premium single-page HTML application using **HTML, CSS, and Vanilla JavaScript**, the project demonstrates how AI can transform difficult decisions into structured, evidence-based insights with a polished commercial-quality interface.

---

## 🚀 Features

- Guided 4-question decision interview
- Interactive decision report dashboard
- Structured comparison of multiple options
- Decision Matrix with weighted scoring
- Assumption & Cognitive Bias Analysis
- Premortem (Failure Prediction)
- 7-Day Decision Validation Plan
- Final AI Verdict with reasoning
- Shareable summary cards
- Premium responsive UI/UX
- Dark mode interface
- Smooth animations and interactive visualizations
- Fully offline HTML application
