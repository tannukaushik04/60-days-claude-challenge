# 📝 Cheat Sheet

## Training Loop

```
Input Data
↓

Prediction
↓

Loss Function
↓

Gradient Calculation
↓

Parameter Update
↓

Repeat Until Convergence
```

## Model Diagnosis

| Training Error | Validation Error | Diagnosis |
|----------------|------------------|------------|
| High | High | Underfitting |
| Low | High | Overfitting |
| Low | Low | Good Fit |

---

## Evaluation Metrics

### Precision

```
TP / (TP + FP)
```

Measures how many predicted positives are actually correct.

---

### Recall

```
TP / (TP + FN)
```

Measures how many actual positives were identified.

---

### F1 Score

```
2 × Precision × Recall
──────────────────────
 Precision + Recall
```

Balances Precision and Recall.
