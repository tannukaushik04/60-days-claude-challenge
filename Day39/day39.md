# Day 39 – PDF Splitter & Merger

## 📌 Overview
Today I built **Foldwork – PDF Splitter & Merger**, a premium browser-based PDF utility that enables users to split, merge, preview, and organize PDF documents entirely on the client side. The application features a modern productivity-focused interface with drag-and-drop uploads, page previews, flexible splitting options, and secure offline processing.

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- Drag & Drop API
- File API
- Canvas Rendering
- Client-side PDF Processing
- Responsive Design
- Offline Single-File Architecture

# 📄 Features Implemented

### PDF Splitter

- Split by custom page ranges
- Split every N pages
- Split after selected pages
- Extract selected pages
- Thumbnail page preview
- Range validation
- Output preview before download

### PDF Merger

- Merge multiple PDF files
- Drag-and-drop file ordering
- Rearrange documents before merging
- Preview merged order
- Export merged PDF

### Additional Features

- Client-side processing (no uploads)
- Offline support
- Drag & Drop interface
- Page thumbnails
- Watermark/Page Number option
- Image extraction
- Compression toggle
- Keyboard shortcuts
- Dark mode UI
- Responsive layout

# 💡 Key Learnings

- Learned how browser-based PDF utilities can securely process files without uploading them to external servers.
- Understood how client-side document manipulation improves privacy and performance.
- Explored drag-and-drop workflows for productivity applications.
- Learned how page thumbnails improve document organization and usability.
- Built an offline-first application using HTML, CSS, and Vanilla JavaScript.
- Experienced designing a commercial-quality interface inspired by modern productivity software.
